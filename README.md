# analyze-runs

Analyze my runs that are exported as gpx files in my Dropbox Folder

[![Build Status](https://travis-ci.org/marcopaga/analyze-runs.svg?branch=master)](https://travis-ci.org/marcopaga/analyze-runs)

## Usage

    $ lein run
    $ lein ring server
